# AI Engineer Learning

Self-learning curriculum for AI Engineer skills, from Python fundamentals to LLM application development.

## Projects

- `currency-converter/` — Command-line currency converter with real-time exchange rates (Week 1-2: Python fundamentals, Git workflow, file I/O, class design)
- `practice/` — Early exercises: type hints, dataclass, dict operations (Week 1)
- `llm-basics/` — LLM API integration and prompt engineering (Week 3+)

## Tech Stack

- Python 3.9
- OpenAI API / Anthropic API
- Requests, Rich
